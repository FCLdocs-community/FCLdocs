从各个网站下载到 FCL 安装包以后，我想各位迫不及待的想安装了

但这里需要注意的是，FCL 安装过程可能会经历曲折，所以请各位小白按以下步骤去安装

1.进入浏览器的下载页面，这里以Chrome浏览器为例：

![FCL文件1](/img/docs/FCL 的下载与安装/FCL 安装/FCL文件1.jpg)
![FCL文件2](/img/docs/FCL 的下载与安装/FCL 安装/FCL文件2.jpg)
![FCL文件3](/img/docs/FCL 的下载与安装/FCL 安装/FCL文件3.jpg)

可以看到，圈出来的文件就是下载好的 FCL 安装包，我们点击它（这里以小米手机为例）：

![FCL安装1](/img/docs/FCL 的下载与安装/FCL 安装/FCL安装1.jpg)
![FCL安装2](/img/docs/FCL 的下载与安装/FCL 安装/FCL安装2.jpg)
![FCL安装3](/img/docs/FCL 的下载与安装/FCL 安装/FCL安装3.jpg)

可以看出，小米手机的 FCL 安装需要点击**两次允许**和**一次继续安装**

![FCL安装4](/img/docs/FCL 的下载与安装/FCL 安装/FCL安装4.jpg)

到了这里，有些小白可能会心里一紧——"我下载的不会是假 FCL 吧？" 

先别急，这里说明一下：**FCL 启动器** 是一款开源软件，所有源代码都在 GitHub 上**公开可查**，不存在任何欺诈行为。

> 毕竟作为一款开源工具，它的定位和社区属性决定了不会去做 [ICP 备案](https://baike.baidu.com/item/ICP%E5%A4%87%E6%A1%88/1220667)（而且备案流程本身也确实比较繁琐）。

跟着图中的箭头和圈出来的选项走，无视风险继续安装。

![FCL安装5](/img/docs/FCL 的下载与安装/FCL 安装/FCL安装5.jpg)
![FCL安装6](/img/docs/FCL 的下载与安装/FCL 安装/FCL安装6.jpg)

等待**FCL 启动器**安装完毕，就可以点击下方的完成了

![FCL安装7](/img/docs/FCL 的下载与安装/FCL 安装/FCL安装7.jpg)
